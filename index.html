Regras do Jogo: Liars Bar

O jogo Liars Bar é um jogo de blefe e coragem, onde os jogadores precisam descartar cartas estratégicas e provar que não estão mentindo, sob o risco de ter que "atirar em si mesmos" caso sejam pegos blefando.

Configuração:

  • Cada jogador recebe um baralho com um conjunto igual de cartas.
  • No início de cada rodada, uma carta é sorteada na mesa para determinar o valor-alvo (por exemplo: "K").
  • Cada jogador possui um revólver fictício, com uma única bala verdadeira.

Regras do Jogo:

  1. No início de cada rodada, uma carta alvo (exemplo: "K") é definida como o valor que todos os jogadores devem descartar.
  2. Os jogadores, em ordem, descartam de 1 a 3 cartas viradas para baixo, anunciando quantas cartas e o valor (exemplo: "Dois Ks").
  3. O próximo jogador pode:
    • Aceitar: A jogada segue para o próximo jogador.
    • Duvidar: As cartas do jogador anterior são reveladas.
  4. Se as cartas reveladas forem verdadeiras (correspondem ao que foi anunciado):
    • O jogador que duvidou deve girar o tambor do revólver e "atirar" em si mesmo.
  5. Se as cartas reveladas forem falsas (não correspondem ao que foi anunciado):
    • O jogador que mentiu deve girar o tambor do revólver e "atirar" em si mesmo.
  6. Se um jogador ficar sem cartas na mão, ele pode "bater" na mesa, declarando sua vitória parcial:
    • O próximo jogador pode escolher "bater" também (caso não tenha cartas) ou desafiar o jogador anterior.
    • Se todos os jogadores optarem por "bater", o último jogador é obrigado a desafiar um dos anteriores.
  7. Após qualquer jogador atirar com o revólver (como consequência de um desafio):
    • Uma nova carta alvo será sorteada na mesa.
    • As cartas serão redistribuídas igualmente entre os jogadores restantes.
  8. O jogo continua até restar apenas um jogador não eliminado.

Obs: Em cada rodada, uma carta vermelha de COPAS pode causar um tiro em todos os jogadores caso seja jogada como verdade e o jogador duvide dela.

Condições de Vitória:

  • O último jogador restante é declarado o vencedor.
  • Se todos forem eliminados exceto um, este ganha o jogo.

Dicas:

  • Seja convincente ao blefar e observe o comportamento dos outros jogadores.
  • Preste atenção às probabilidades: quantas cartas do valor-alvo ainda podem estar em jogo?
  • Use as dúvidas estrategicamente para pressionar jogadores que parecem estar blefando.

Clique no botão de cada jogador para puxar o gatilho.
